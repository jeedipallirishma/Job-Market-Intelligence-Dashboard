# Job Market Intelligence Dashboard

## Overview

This project analyzes job market trends using Python and Exploratory Data Analysis (EDA).

The goal is to identify:

* Most demanded skills
* Top hiring locations
* Top hiring companies
* Industry-wise demand
* Job title demand
* Company rating distribution

## Tools & Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

## Dataset

Data Analyst Job Postings Dataset

## Key Analysis

### Skill Demand Analysis

Analyzed demand for:

* Python
* SQL
* Excel
* Power BI
* Tableau
* Machine Learning

### Hiring Locations

Identified the cities with the highest number of job opportunities.

### Hiring Companies

Analyzed companies with the most active job postings.

### Industry Analysis

Explored industries generating the highest demand for data professionals.

### Rating Distribution

Studied company rating patterns across job postings.

## Project Outcomes

* SQL and Python emerged as highly demanded skills.
* Certain cities dominated hiring activity.
* Industry-wise demand trends were identified.
* Company ratings were analyzed to understand employer reputation.

## Future Improvements

* Automated Skill Recommendation System
